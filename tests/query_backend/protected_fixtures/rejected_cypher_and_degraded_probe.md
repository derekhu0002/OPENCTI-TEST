# Query Backend 保护夹具

本文件冻结查询后端显性 testcase 的 GIVEN 与 WHEN，不允许在后续编码阶段改写为其它场景。

## GIVEN

1. 远端 AI Agent 只能通过查询后端访问图副本，不直接持有 Neo4j 或 OpenCTI 高权限凭证。
2. 查询后端已经启用只读强制、schema 白名单、预算控制、审计日志和 freshness 状态暴露。
3. 正常场景下，副本可用且新鲜度处于可接受范围内，后端可以返回可继续消费的图结果。
4. 降级场景下，副本不可达、同步故障或新鲜度超阈值。

## WHEN

1. Agent 提交一条受后端允许执行的只读图查询。
2. Agent 提交包含写操作或明显越界特征的 Cypher。
3. Agent 在副本不可安全使用时提交需要多跳遍历的图查询。
